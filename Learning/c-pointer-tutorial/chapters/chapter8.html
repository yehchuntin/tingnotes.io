Character arrays and pointers - part 1

字元陣列與指標基本概念與操作

1 字串基本概念

在 C 語言中，字串是以字元陣列的形式儲存，並以 '\0' 作為字串結束的標記。以下列出幾個重要特性：

💡 提醒

字串必須以 '\0' 結尾；宣告字元陣列時其大小需包含末端的 '\0'；也可以使用字串常數或逐個字元初始化陣列。

  • 字串必須以 '\0' 結尾，稱為 null terminator。
  • 宣告字元陣列時，其大小需包含末端的 '\0'。
  • 可以用字串常數或逐個字元初始化。

下面的範例展示三種宣告與初始化字串的方法：

// string_declarations.c
#include <stdio.h>
#include <string.h>

int main() {
    // 方法1: 自動分配空間
    char str1[] = "Mia and Ting";
    
    // 方法2: 指定陣列大小
    char str2[20] = "Hello";
    
    // 方法3: 逐個字元初始化
    char str3[6] = {'H','e','l','l','o','\0'};
    
    printf("str1: %s\n", str1);
    printf("str2: %s\n", str2);
    printf("str3: %s\n", str3);
    
    return 0;
}

宣告完字串後，常見的兩個函式 sizeof() 和 strlen() 有何差異？觀察以下程式：

// sizeof_vs_strlen.c
#include <stdio.h>
#include <string.h>

int main() {
    char str[] = "Mia and Ting";
    
    printf("字串內容: \"%s\"\n", str);
    printf("sizeof(str) = %lu bytes\n", sizeof(str));
    printf("strlen(str) = %lu characters\n", strlen(str));
    printf("\n說明：\n");
    printf("- sizeof 包含 null terminator '\\0'\n");
    printf("- strlen 只計算實際字元數量\n");
    
    return 0;
}
函數 回傳值 計算時機 是否包含 '\0' 適用場合
sizeof() 陣列總大小 編譯時期 是 記憶體分配、陣列大小計算
strlen() 字串長度 執行時期 否 字串長度檢查、循環控制

2 字串與指標

字串名稱本身就是指向第一個字元的指標，因此可以利用指標來操作和遍歷字串。指標讓字串處理更加靈活：

💡 提醒

字串變數其實是指向首元素的指標常量；可利用指標運算（如 ptr++）走訪字串；陣列索引與指標運算可以互換使用。

  • 字串變數其實是指向首元素的指標常量。
  • 指標運算（如 ptr++）可用來走訪字串。
  • 陣列索引與指標運算可以互換使用。

以下示範指標如何存取及修改字串內容：

// string_pointer_basic.c
#include <stdio.h>
#include <string.h>

int main() {
    char str[6] = "Hello";
    char *ptr;
    
    printf("原始字串: %s\n", str);
    printf("指標大小: %lu bytes\n", sizeof(ptr));
    
    ptr = str;  // ptr 指向 str 的第一個元素
    
    printf("透過指標存取:\n");
    printf("ptr[0] = %c\n", ptr[0]);
    printf("ptr[1] = %c\n", ptr[1]);
    printf("*(ptr+2) = %c\n", *(ptr+2));
    
    // 修改字串內容
    ptr[0] = 'A';
    printf("修改後: %s\n", str);
    
    return 0;
}

也可以使用指標運算來遍歷整個字串：

// string_traversal.c
#include <stdio.h>

int main() {
    char str[] = "Programming";
    char *ptr = str;
    
    printf("使用指標遍歷字串:\n");
    
    // 方法1: 陣列索引
    printf("方法1 - 陣列索引: ");
    for(int i = 0; str[i] != '\0'; i++) {
        printf("%c", str[i]);
    }
    printf("\n");
    
    // 方法2: 指標運算
    printf("方法2 - 指標運算: ");
    while(*ptr != '\0') {
        printf("%c", *ptr);
        ptr++;
    }
    printf("\n");
    
    return 0;
}

3 字串傳入函式

當字串作為參數傳入函式時，字元陣列名稱會自動退化為指標。函式可以接收 char* 或 char[] 參數，並在內部修改字串內容。以下範例示範自訂印字函式： 

// custom_string_func.c
#include <stdio.h>
#include <string.h>

void print(char *C) {
    // 以指標接收字串
    printf("%s\n", C);      // 印出整個字串
    int i = 0;
    while (C[i] != '\0') {  // 逐字元列印
        printf("%c", *(C+i)); // same as C[i]
        i++;
    }
    printf("\n");
}

int main() {
    char C[6] = "Hello";
    print(C);               // 傳入字元陣列 → 自動退化為指標
    return 0;
}

最後範例展示 strcpy、strcat、strcmp 等常用字串處理函式的使用方式：

// standard_string_funcs.c
#include <stdio.h>
#include <string.h>

int main() {
    char str1[50] = "Hello";
    char str2[] = " World";
    char str3[50];
    
    printf("原始字串: \"%s\"\n", str1);
    
    // 字串複製
    strcpy(str3, str1);
    printf("複製後: \"%s\"\n", str3);
    
    // 字串串接
    strcat(str1, str2);
    printf("串接後: \"%s\"\n", str1);
    
    // 字串比較
    if(strcmp(str1, "Hello World") == 0)
        printf("str1 與 \"Hello World\" 相同\n");
    else
        printf("str1 與 \"Hello World\" 不同\n");
    
    // 其他函式：strlen()
    printf("str1 長度: %lu\n", strlen(str1));
    
    return 0;
}

✅ 本章重點總結

🎯 核心概念

字串本質上是字元陣列，但其運作高度依賴指標與記憶體操作，理解 '\0' 的作用是避免記憶體錯誤的關鍵。

  • 字串在 C 中是字元陣列，以 '\0' 作為結束符號。
  • 使用指標可以更加靈活地讀取和修改字串內容。
  • 函式接受字串參數時，陣列名稱會退化為指標，可以直接在函式內部改變字串。
  • sizeof 和 strlen 回傳的結果不同：前者包含結束字元，後者僅計算字串長度。
