Pointers to Pointers

建立指標鏈、逐層解參考、位址關係、與改值路徑

1 建立一層一層的指標鏈

從一般變數 x 出發，讓 p 指向 x、q 指向 p、r 指向 q，形成 x ← p ← q ← r 的鏈。 

// pointer_to_pointer_demo.c
#include <stdio.h>

int main(void) {
    int  x = 5;        // 一般整數
    int *p = &x;       // p 指向 x
    *p = 6;            // 透過 p 改 x → x = 6

    int **q = &p;      // q 指向 p
    int ***r = &q;     // r 指向 q

    printf("x     = %d\n", x);
    printf("&x    = %p\n", (void*)&x);

    printf("p     = %p  (should equal &x)\n", (void*)p);
    printf("&p    = %p\n", (void*)&p);
    printf("*p    = %d  (value of x)\n\n", *p);

    printf("q     = %p  (should equal &p)\n", (void*)q);
    printf("&q    = %p\n", (void*)&q);
    printf("*q    = %p  (should equal p)\n", (void*)*q);
    printf("**q   = %d  (value of x via q)\n\n", **q);

    printf("r     = %p  (should equal &q)\n", (void*)r);
    printf("&r    = %p\n", (void*)&r);
    printf("*r    = %p  (should equal q)\n", (void*)*r);
    printf("**r   = %p  (should equal p)\n", (void*)**r);
    printf("***r  = %d  (value of x via r)\n\n", ***r);

    // 透過 r 直接改到 x
    ***r = 10;
    printf("after ***r = 10 -> x = %d\n", x);

    // 透過 q 計算並寫回 x
    **q = *p + 2;  // x = 12
    printf("after **q = *p + 2 -> x = %d\n", x);

    return 0;
}

2 逐層解參考（觀念上像穿隧道）

每多一顆星號 *，就多走一層：*p 取 x、**q 也取 x、***r 仍然到 x。但途徑不同。 

printf("*p   = %d\n", *p);      // 直達 x
printf("**q  = %d\n", **q);     // 經由 q → p → x
printf("***r = %d\n", ***r);    // 經由 r → q → p → x
🧭 記憶口訣

星號數 = 要走幾層 間接層級 才能到達最終變數的位址鏈末端。

3 逐層的「位址」關係

位址之間存在等式關係：p == &x、q == &p、r == &q、以及 *q == p、**r == p 等。 

printf("p  = %p, &x = %p\n", (void*)p,  (void*)&x);
printf("q  = %p, &p = %p\n", (void*)q,  (void*)&p);
printf("r  = %p, &q = %p\n", (void*)r,  (void*)&q);
printf("*q = %p,  p = %p\n", (void*)*q, (void*)p);

用 %p 印位址，常見等式會在執行時成立（數值不同但相等關係相同）。

4 從不同「改值」路徑，都會打到同一個 x

因為 *p、**q、***r 都對準同一個 x，所以從任何路徑寫入，結果都會一致。 

*p   = 18;          // 直接改 x
**q  = 6;           // 經由 q 改 x
***r = *p + 2;     // 經由 r 改 x
📌 重點

程式碼的「路徑」或變數名稱可以不同，但最後都落在同一個記憶體位置（x）。

5 如何判斷要解幾層？

  • 先看 你擁有的是什麼型別：int*？int**？int***？
  • 再看你想要的目標型別：是 int 的值，還是 int* 的位址？
  • 每多一顆星號代表「沿指標再往下一層」— 直到拿到想要的型別為止。
// 例：從 r (int ***) 要拿到 x (int)
// 路徑： r → *r (q) → **r (p) → ***r (x)
int value = ***r;  // 目標型別是 int，所以解三層

✅ 本章重點總結

  • 指標鏈：x ← p ← q ← r，層層相扣。
  • 解參考層級：星號數量 = 沿鏈下鑽的層數。
  • 位址關係：p == &x、q == &p、r == &q；*q == p、**r == p。
  • 改值路徑：從 *p、**q、***r 任一路徑寫入，最後都會改到同一個 x。
